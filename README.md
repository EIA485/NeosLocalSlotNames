# LocalSlotNames

A [BepisLoader](https://github.com/ResoniteModding/BepisLoader) mod for [Resonite](https://resonite.com/) that shows the local name of a slot in the hierarchy. 

Fixes neos issue [#3515](https://github.com/Neos-Metaverse/NeosPublic/issues/3515)

## Installation
1. Install [BepisLoader](https://github.com/ResoniteModding/BepisLoader).
1. Place [LocalSlotNames.dll](https://github.com/eia485/NeosLocalSlotNames/releases/latest/download/LocalSlotNames.dll) into your `plugins` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\BepInEx\plugins` for a default install.
1. Start the game. If you want to verify that the mod is working you can check your BepInEx log at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\BepInEx\LogOutput.log`
